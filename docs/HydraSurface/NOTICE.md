# NOTICE — scope, interoperability and trademarks

**HydraSurface** is an **original, independent** interoperability software
implementation. Its purpose is to let a mixing console act as a **control surface**
for a DAW, translating between control protocols (HiQnet and Mackie HUI) over
network and MIDI.

## What this project does NOT include
- No third‑party firmware, firmware image, binary, disassembly or teardown of any
  third‑party product.
- No third‑party SDK, source code or proprietary material.
- Only the author's original code and **protocol facts** (message formats, parameter
  identifiers, port numbers) necessary for interoperability.
